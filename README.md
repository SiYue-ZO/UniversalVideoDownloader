# 全能网页视频下载器 (Universal Video Downloader)

基于 Qt C++ 编写的跨平台图形化视频下载工具。底层强力驱动依赖 `yt-dlp` 和 `FFmpeg`，支持解析并下载各大视频网站（如 B站、YouTube 等）的高清视频流，并支持合并多轨道音视频。

## ✨ 核心特性

* **跨平台支持**：支持 macOS 本地编译运行，以及通过 GitHub Actions 云端全自动编译 Windows 可执行程序。
* **智能解析**：输入网址即可自动逆向提取真实的音视频流，并提供友好的画质分辨率下拉菜单供用户选择。
* **防爬虫绕过 (Cookie 注入)**：内置浏览器 Cookie 读取功能（支持 Firefox, Chrome, Edge, Safari）。对于 B站 等需要登录才能解锁 1080P/大会员画质的网站，只需在本地浏览器登录账号，程序即可无缝借用身份凭证完成高清下载。
* **多线程与异步界面**：底层使用 `QProcess` 与 `QtConcurrent`，解析与下载过程绝不卡死图形界面，并附带实时进度条与运行日志。

🪟 Windows 使用指南
Windows 用户无需配置任何 C++ 或 Qt 编译环境，利用云端打包好的文件即可直接运行。
1. 获取主程序
	1.	进入本 GitHub 仓库的 [Actions] 页面。
	2.	找到最近一次绿色的 Windows Build 任务。
	3.	滑到底部 Artifacts 区域，下载压缩包并解压。
2. 准备底层引擎 (必须步骤)
为了让程序能正常抓取和合并视频，你需要手动下载两个引擎文件，并把它们和主程序放在同一个文件夹里：
•	下载 yt-dlp.exe (底层解析下载器)
•	下载 FFmpeg.exe (音视频合并工具)
3. 运行程序
双击运行解压出的 UniversalVideoDownloader.exe 即可。
🍪 关于绕过 B站/平台 防爬虫的特别说明
很多网站会对未登录请求进行拦截，或强制降级为 480P。请在界面上选择你正在使用的浏览器以注入 Cookie：
	1.	Firefox (强烈推荐)：火狐浏览器的 Cookie 存储无强系统级加密，跨平台兼容性最佳，解析速度极快且极少报错。
	2.	Chrome / Edge：Chromium 内核会与系统凭据管家绑定。在抓取时，系统可能会弹出安全提示，需要你输入电脑的开机密码或进行指纹验证。
	3.	Safari (仅限 macOS)：苹果的沙盒权限极严。如果选择 Safari，你必须去 macOS 的 系统设置 -> 隐私与安全性 -> 完全磁盘访问权限 中，勾选本程序，否则程序会因为无权读取 Safari 数据而报错。
基本逻辑：在程序里选择某个浏览器前，请确保你已经在该浏览器中打开并登录了目标视频网站。
